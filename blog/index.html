DRAFT :: 使用hugo创建Bolg

#第一步:安装hugo 参考：https://gohugo.io/getting-started/installing/ ##1.在windows中安装hugo,先安装Chocolatey(windows包管理) 参考：https://chocolatey.org/install 利用cmd安装的命令 @“%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe” -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command “iex ((New-Object System.Net.WebClient).DownloadString(‘https://chocolatey.org/install.ps1'))" && SET “PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin”

    利用PowerShell安装的命令
        Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

    注:
        1.我也是第一次在windows中安装管理，我选择了chocolatey，也有其他的包管理软件。
        2.安装的时候要对应终端，不行就自行百度。
##2.利用Chocolatey安装hugo
    命令:choco install hugo -confirm        

    查看版本命令:hugo version

#第二步:用hugo生成博客 ##1.生成站点 hugo new site myblog(“myblog是我取的文件名,可任意取名”) ##2.进入站点 cd myblog

#第三步:下载并设置主题 主题网站:https://themes.gohugo.io/ ##1.下载主题需要通过git,所以要先下载安装git ##2.主题下载有时会下载到myblog目录下的themes文件夹中,有时不会,不会的下载时就在终端中先进入themes文件夹中再下载 #第四步:在本地启动个人博客
命令:hugo server -t m10c –buildDrafts hugo server 开启hugo服务 -t 确定主题
m10c 主题名

    命令输入后访问 http://localhost:1313/ 
    这是默认生成的,输入命令后能在下面的一串字符中找到,如果1313端口被占用,会随机生使用别的端口

#第五步:写一篇文章 命令:hugo new post/blog.md

    创建post/blog.md文件,blog文件名,随便取,md是markdown格式的后缀名

    创建完成后 post/blog.md在myblog/context目录下 

    修改blog.md 

    在本地启动博客 
        hugo server -t m10c --buildDrafts 
    访问 
    命令输入后访问 http://localhost:1313/

#第六步:部署到github 这里不介绍了
